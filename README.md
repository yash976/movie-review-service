# 🎬 Movie Review Microservice

A Spring Boot microservice for managing movies and reviews.

## 🚀 Features
- Add & fetch movies
- Add & fetch reviews
- RESTful APIs
- Clean architecture

## 🛠 Tech Stack
- Java
- Spring Boot
- MySQL
- JPA / Hibernate

## 📌 API Endpoints

### Movies
- POST /movies
- GET /movies
- GET /movies/{id}

### Reviews
- POST /reviews
- GET /reviews/movie/{movieId}

## ▶️ Run Locally

```bash
mvn spring-boot:run
